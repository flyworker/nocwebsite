Summary

Currently building an anycast DNS service spanning multiple countries and continents.

  • Unlimited domains
  • IPv4 and IPv6
  • Supports NOTIFY
  • DNSSEC compliant
  • AXFR over both v4 + v6

  • Full REST API access to all configuration
  • Coming soon:
    • Web interface
    • TSIG inbound
    • Multi master

Configuration

Secondary DNS zones are served via anycast from (please add this to your NS record and delegate at your registrar):

  • ns1.esgob.com
  • 193.47.147.100
  • 2001:67c:1b43:100::100

We operate a dedicated AXFR host, please allow AXFR zone transfers from:

  • axfr.esgob.com
  • 195.177.253.142
  • 2001:67c:1b40:142::142
We will honor notifies that are sent to the AXFR host.
