API Documentation

Summary

Our API is accessible to anyone who has an account. It consists of a set of callable methods and some API endpoints.

It is REST based and capable of returning either JSON (by default) or XML.

Resources

Resource Authentication Description
GET /1.0/accounts.get Account Return account status; credit balance, etc
GET /1.0/domains.list Account Returns all hosted domains
GET /1.0/domains.slaves.list Account Returns all hosted slave domains
GET /1.0/domains.slaves.add Account Adds a new slave domain
GET /1.0/domains.slaves.delete Account Deletes a slave domain
GET /1.0/domains.slaves.forcetransfer Account Force AXFR / transfer from master of a slave domain
GET /1.0/domains.slaves.updatemasterip Account Updates the master IP of a slave domain
GET /1.0/domains.slaves.axfrout.add Account Add a host allowed to AXFR out
GET /1.0/domains.slaves.axfrout.delete Account Delete a host allowed to AXFR out
GET /1.0/domains.tools.soacheck Account Retrieve the domain SOA serial number from the master and each anycast node
